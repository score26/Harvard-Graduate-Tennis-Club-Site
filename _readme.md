# Harvard Graduate Tennis Club Website

Static HTML/CSS site hosted on GitHub Pages at harvardgraduatetennis.org

## Files
- index.html — homepage
- leadership.html — officer directory
- officers/ — headshot photos (800x1067px JPEG)
- images/ — other site photos

## How to edit
Edit files directly in GitHub's browser editor and commit. Changes go live within 60 seconds.

## Design
- Font: Archivo (Google Fonts)
- Colors: Beige #EAE7E2, Crimson #A51C30, Sage #818C7E
- Officer photos: 800x1067px, compressed via Squoosh MozJPEG quality 75
